# Engineering Culture

## Core Values

1. **Unwavering integrity**
2. **Trailblazing creativity**
3. **Speed is our superpower**

## Engineering Philosophy

- **Work fast, make mistakes** — speed and meeting deadlines matters more than perfection. Engineers should feel free to make mistakes and iterate.
- **Vibe code encouraged** — engineers are encouraged to use AI heavily. The goal is to be code reviewers, not coders.
- **AI-first** — Claude is the preferred AI tool. Everyone should be leveraging it.
- **Account context matters** — engineers need context on the full client account, not just their current task. Account context enables autonomous decision-making.
- **Quality output** — no specific coding style preferences. What matters is sustainable, quality output for the customer long-term. Done right and done fast.

## What to Optimize For

- Speed of delivery
- Sustainable quality for the customer
- Engineer autonomy through context
- AI-assisted development as the default
