# Deploy Platforms

No standard deploy target — varies by project. Check the project's repo or CLAUDE.md for specifics.

| Platform | Used For |
|----------|----------|
| **Render** | Flask/Python web apps |
| **Google Cloud (Cloud Run)** | TypeScript/React apps (e.g., DCC) |
| **Odoo.sh (Odoo Cloud)** | ERP systems (e.g., S40S Odoo) |
| **SiteGround** | WordPress / WooCommerce sites |
| **Lovable + Supabase** | Simple AI-built applications, rapid prototypes |
