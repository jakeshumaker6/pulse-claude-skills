# Client Handoffs and Communication

## Project Handoff

- **Project Acceptance Agreement** — sent at project conclusion, client signs to confirm work is complete and ready for handoff

## Client Call Cadence

- **>$30K engagements:** Weekly calls (default)
- **Smaller engagements:** Every other week or monthly, depending on agreement
- **All client calls must be recorded with Grain**

## Communication Channels

| Context | Channel |
|---------|---------|
| Internal team communication | Slack |
| Project-related questions | Slack |
| Task-specific discussion | ClickUp comment on the task |
| Client communication | Gmail / client calls |
| Client calls | Always recorded with Grain |
| AI tool | Claude (preferred, team-wide) |

## Tool Stack

- **Slack** — internal comms
- **ClickUp** — project management + internal documentation
- **Gmail / Google Workspace** — external comms + external document storage
- **Grain** — meeting recording and notes
- **Claude** — preferred AI tool (team-wide)
- **Wispr Flow** — voice-to-text productivity
